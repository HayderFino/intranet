# Portal de Intranet - CAS

Este repositorio contiene el código fuente y los activos del Portal de Intranet de la Corporación Autónoma Regional de Santander (CAS).

## Estructura de Documentación

Toda la documentación detallada del proyecto se encuentra centralizada en la carpeta `documentacion/`:

1.  **[Guía General](general.md)**: Visión global y estructura del repositorio.
2.  **[Documentación Técnica](tecnico.md)**: Instalación, ejecución y control de versiones.
3.  **[Arquitectura MVC](arquitectura_mvc.md)**: Detalle del patrón de diseño en el backend.
4.  **[Gestión de Datos](gestion_datos.md)**: Organización de archivos y activos (data).
5.  **[Administración](administracion.md)**: Guía del panel de gestión.
6.  **[Herramientas](herramientas.md)**: Catálogo de utilidades y páginas especiales.

---

## Inicio Rápido

```powershell
cd practicas
npm install
node server.js
```
Acceso: `http://localhost:3000`
