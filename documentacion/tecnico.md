# Documentación Técnica

Este documento detalla los aspectos técnicos necesarios para la ejecución y mantenimiento del portal.

## 🚀 Instalación y Ejecución

El portal funciona bajo un entorno de Node.js con el framework Express.

### Requisitos
- Node.js (v14+)
- npm

### Pasos
1. Abrir terminal en `c:\Users\HAYDER\Documents\intranet\intranet\practicas`.
2. Ejecutar `npm install` para instalar dependencias (`express`, `cors`, `multer`).
3. Iniciar con `node server.js`.
